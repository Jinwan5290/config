# @fbi-js/eslint-config

ESLint base config.

## Usage

```bash
npm i -D @fbi-js/eslint-config
```

`./package.json` (you can also use other [configuration files](https://eslint.org/docs/user-guide/configuring#using-configuration-files-1))

```json
{
  "eslintConfig": {
    "extends": [
      "@fbi-js"
    ]
  }
}
```
