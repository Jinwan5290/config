# @fbi-js/stylelint-config

StyleLint config.

## Usage

```bash
npm i -D @fbi-js/stylelint-config
```

`./package.json` (you can also use other [configuration files](https://stylelint.io/user-guide/configure))

```json
{
  "stylelint": {
    "extends": [
      "@fbi-js/stylelint-config"
    ]
  }
}
```
